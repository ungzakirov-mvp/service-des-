

# Novum Tech — Полный редизайн IT-аутсорсинг сайта

## Концепция
Премиальный B2B сайт для IT-аутсорсинговой компании в Ташкенте. Тёмная цветовая схема с акцентами, минималистичный корпоративный стиль, чёткая типографика, плавные микро-анимации.

## Дизайн-система
- **Цвета**: Тёмный фон (#0A0A0F), белый текст, акцент — синий/голубой (#3B82F6)
- **Типографика**: Чёткая иерархия H1-H3, Inter/системный шрифт
- **Скругления**: 8-12px для карточек, 6px для кнопок
- **Анимации**: Fade-in при скролле, hover-эффекты на карточках и кнопках

## Структура страницы (Single Page с якорями)

### 1. Навигация (sticky)
- Логотип Novum Tech (текстовый)
- Пункты: Услуги / О нас / Как мы работаем / Контакты
- CTA кнопка «Запросить аудит»
- Мобильное гамбургер-меню с плавным выездом

### 2. Hero-секция
- Заголовок: «IT-аутсорсинг для бизнеса в Ташкенте»
- Подзаголовок: «Берём полную ответственность за IT-инфраструктуру: стабильность, безопасность и контроль для компаний 20–100 сотрудников»
- Две CTA кнопки: «Запросить IT-аудит» и «Узнать больше»
- Фоновый градиент с геометрическими элементами

### 3. Trust Indicators
- Горизонтальная полоса с ключевыми метриками: «SLA 99.9%», «20–100 сотрудников», «Реакция до 30 мин», «Прозрачная отчётность»

### 4. Услуги (3 карточки)
- **Server Management** — Надёжные серверные решения
- **IT Security Audit** — Защита и безопасность IT-инфраструктуры
- **Workstation Support** — Поддержка IT-оборудования
- Каждая карточка с иконкой, описанием и кнопкой «Подробнее»
- Hover-эффект с подъёмом и тенью

### 5. О компании
- Краткое описание Novum Tech
- Ключевые преимущества в виде списка с иконками (SLA, прозрачность, оперативность, экспертиза)

### 6. Как мы работаем (4 шага)
- Визуальный таймлайн/степпер
- 1: Экспресс-аудит IT → 2: Формируем предложение → 3: Подключаем и настраиваем → 4: Поддержка и контроль
- Каждый шаг с номером, заголовком и описанием

### 7. FAQ секция
- Аккордеон с 5-6 частыми вопросами об IT-аутсорсинге
- Плавная анимация раскрытия

### 8. Форма «Запросить IT-аудит»
- Поля: Имя, Компания, Email, Телефон, Комментарий
- Валидация полей с сообщениями об ошибках
- Уведомление об успешной отправке (toast)
- Данные сохраняются в localStorage как демо (без бэкенда)

### 9. Footer
- Логотип и краткое описание
- Навигационные ссылки
- Контактная информация (телефон, email, адрес в Ташкенте)
- Копирайт © 2026 Novum Tech

## Адаптивность
- Desktop: полноразмерный layout с grid
- Tablet: перестройка сетки услуг в 2 колонки, затем в 1
- Mobile: гамбургер-меню, вертикальный layout, увеличенные touch-зоны для кнопок

## Анимации
- Fade-in секций при скролле (CSS-анимации через Intersection Observer)
- Hover-подъём карточек услуг
- Плавный scroll при навигации по якорям

## SEO
- Корректная иерархия заголовков H1→H2→H3
- Meta title и description
- Семантическая HTML-разметка
- Alt-тексты для всех визуальных элементов
